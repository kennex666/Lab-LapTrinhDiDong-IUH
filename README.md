## Mô Tả Dự Án

Dự án này là một bài tập lập trình di động được xây dựng bằng React Native. Dự án nhằm mục đích rèn luyện kỹ năng phát triển ứng dụng di động với React Native, bao gồm các chức năng cơ bản và nâng cao như giao diện người dùng, quản lý trạng thái, và tương tác với API.

## Hướng Dẫn Cài Đặt

Để chạy dự án này trên máy tính của bạn, hãy làm theo các bước sau:

### 1. Cài Đặt Node.js và npm

Đảm bảo rằng bạn đã cài đặt Node.js và npm (Node Package Manager) trên máy tính của bạn. Bạn có thể tải Node.js từ [trang chính thức](https://nodejs.org/) và npm sẽ được cài đặt cùng với Node.js.

### 2. Cài Đặt Expo CLI (Tùy chọn)

Nếu bạn muốn sử dụng Expo để phát triển và kiểm tra ứng dụng, hãy cài đặt Expo CLI bằng cách chạy lệnh sau:

```bash
npm install -g expo-cli
```
### 3. Clone Repository
Sao chép repo về máy tính của bạn bằng lệnh:

```bash
git clone https://github.com/username/repository-name.git
cd repository-name
```

### 4. Cài Đặt Các Gói Phụ Thuộc
Cài đặt các gói phụ thuộc của dự án bằng cách chạy lệnh:

```bash
npm install
```


### 5. Chạy Ứng Dụng
Để chạy ứng dụng trên thiết bị mô phỏng hoặc thiết bị thật, sử dụng Expo CLI:

```bash
npm start
```
